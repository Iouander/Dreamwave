You can place any .WAV music file in here and it will play within any menu. 
It cycles through them in order, so you can use numbers in the file names to specify their order.

You can cycle them within the game as well.

Keep the files in the WAV format. If you need a converter, (https://cloudconvert.com/mp3-to-wav - https://www.freeconvert.com/mp3-to-wav/) - direct links.
